## 시작하기

### 웹사이트 접속
- **라이브 사이트**: https://guild-association.vercel.app
- **지원 브라우저**: Chrome, Firefox, Safari, Edge (최신 버전)

### 첫 화면
처음 접속하면 다음과 같은 요소들이 보입니다:
- 상단 네비게이션 바
- "Login with GitHub" 버튼
- 퀘스트 목록 (로그인 필요)

## 회원가입 및 로그인

### GitHub 계정으로 로그인하기

1. **로그인 버튼 클릭**
   - 화면 우측 상단의 **"Login with GitHub"** 버튼을 클릭합니다.

2. **GitHub 인증**
   - GitHub 로그인 페이지로 리디렉션됩니다.
   - GitHub 계정 정보를 입력합니다.
   - 이미 로그인된 상태라면 자동으로 넘어갑니다.

3. **권한 승인**
   - Guild Association이 요청하는 권한을 확인합니다.
   - 요청 권한:
     - 사용자 프로필 정보 (이름, 아이디)
     - 이메일 주소 (선택)
   - **Authorize** 버튼을 클릭합니다.

4. **자동 회원가입**
   - 처음 로그인하는 경우 자동으로 계정이 생성됩니다.
   - 초기 레벨: 1
   - 초기 경험치: 0
   - GitHub 프로필 정보가 자동으로 등록됩니다.

5. **로그인 완료**
   - 대시보드로 리디렉션됩니다.
   - 우측 상단에 사용자 이름이 표시됩니다.

### 로그아웃

1. 우측 상단의 **사용자 이름** 클릭
2. **"Logout"** 버튼 클릭
3. 로그아웃 완료 - 메인 페이지로 이동

## 대시보드 사용법

### 화면 구성

#### 상단 네비게이션 바
- **Guild Association** (로고) - 클릭 시 대시보드로 이동
- **Leaderboard** - 리더보드 페이지로 이동
- **사용자 정보**:
  - 레벨 표시
  - 사용자 이름
  - Logout 버튼

#### 대시보드 메인 영역
- **환영 메시지**: "Welcome, [사용자이름]!"
- **퀘스트 목록**: 카드 형식으로 표시
  - 퀘스트 제목
  - 난이도 (EASY, MEDIUM, HARD)
  - 경험치 보상
  - 설명

### 퀘스트 카드 읽는 법

```
┌─────────────────────────────────┐
│ 📜 퀘스트 제목                    │
├─────────────────────────────────┤
│ Difficulty: MEDIUM              │
│ Reward: 200 XP                  │
├─────────────────────────────────┤
│ 퀘스트 설명...                   │
│                                 │
│ [View Details] 버튼             │
└─────────────────────────────────┘
```

## 퀘스트 시스템

### 퀘스트 난이도

| 난이도 | 경험치 보상 | 특징 |
|--------|-------------|------|
| **EASY** | 100 XP | 간단한 작업, 초보자용 |
| **MEDIUM** | 200 XP | 중간 난이도, 일반적인 작업 |
| **HARD** | 500 XP | 어려운 작업, 숙련자용 |

### 퀘스트 완료하기

#### 1단계: 퀘스트 선택
- 대시보드에서 원하는 퀘스트 카드를 찾습니다.
- **"View Details"** 버튼을 클릭합니다.

#### 2단계: 퀘스트 상세 정보 확인
퀘스트 상세 페이지에서 다음 정보를 확인할 수 있습니다:
- 전체 설명
- 난이도
- 경험치 보상
- 제출 폼

#### 3단계: 퀘스트 수행
- 퀘스트 요구사항을 읽고 작업을 수행합니다.
- 필요한 경우 코드 작성, 문서 작성 등을 진행합니다.

#### 4단계: 제출
1. 제출 폼에 내용을 입력합니다.
   - 작업 내용 설명
   - 완료한 결과물 링크 (선택)
   - GitHub PR 링크 (해당되는 경우)

2. **"Submit Quest"** 버튼을 클릭합니다.

3. 제출 확인 메시지가 표시됩니다.

#### 5단계: 보상 받기
- 제출이 승인되면 자동으로 경험치가 지급됩니다.
- 레벨업 조건을 충족하면 자동으로 레벨이 올라갑니다.
- 화면에 보상 알림이 표시됩니다.

### 제출 상태 확인
- 프로필 페이지에서 제출한 퀘스트 목록을 확인할 수 있습니다.
- 완료된 퀘스트는 "Completed" 표시가 됩니다.

## 레벨 및 경험치 시스템

### 레벨 시스템

```
레벨 1 → 레벨 2: 1,000 XP 필요
레벨 2 → 레벨 3: 1,000 XP 필요
레벨 3 → 레벨 4: 1,000 XP 필요
...
```

- 각 레벨업에 **1,000 XP** 필요
- 레벨이 올라가면 프로필에 표시됩니다.
- 리더보드 순위에 영향을 줍니다.

### 경험치 획득 방법

1. **퀘스트 완료**:
   - EASY 퀘스트: +100 XP
   - MEDIUM 퀘스트: +200 XP
   - HARD 퀘스트: +500 XP

2. **중복 완료 가능**:
   - 같은 퀘스트를 여러 번 완료할 수 있습니다.
   - 매번 경험치를 받습니다.

### 경험치 확인
- 우측 상단 네비게이션 바에 현재 레벨 표시
- 프로필 페이지에서 총 경험치 확인 가능

## 리더보드

### 리더보드 접속
1. 상단 네비게이션 바에서 **"Leaderboard"** 클릭
2. 전체 사용자 순위 확인

### 순위 기준
- **1순위**: 레벨 (높을수록 상위)
- **2순위**: 경험치 (레벨이 같으면 경험치로 비교)

### 리더보드 정보
각 사용자별로 다음 정보가 표시됩니다:
- **순위**: 1위, 2위, 3위...
- **사용자 이름**: GitHub 사용자명
- **레벨**: 현재 레벨
- **경험치**: 총 경험치
- **완료한 퀘스트 수**: 제출 및 완료한 퀘스트 개수

### 순위 표시
- 🥇 **1위**: 금메달
- 🥈 **2위**: 은메달
- 🥉 **3위**: 동메달
- 4위 이하: 숫자로 표시

1. 우측 상단의 **사용자 이름** 클릭

## 문제 해결

### 로그인이 안 됩니다
**증상**: "Login with GitHub" 버튼을 클릭해도 로그인되지 않음

**해결 방법**:
1. 브라우저의 쿠키 설정 확인
   - 설정 > 개인정보 및 보안 > 쿠키
   - "타사 쿠키 허용" 활성화

2. 시크릿/프라이빗 모드에서 테스트
   - Ctrl+Shift+N (Chrome) 또는 Ctrl+Shift+P (Firefox)
   - 확장 프로그램이 간섭하는지 확인

3. 브라우저 캐시 삭제
   - Ctrl+Shift+Delete
   - 쿠키와 캐시 삭제
   - 페이지 새로고침

4. 다른 브라우저로 시도

### 로그인했는데 계속 로그인 버튼이 보입니다
**증상**: GitHub 인증은 완료됐지만 웹사이트에서 로그인 상태가 유지되지 않음

**해결 방법**:
1. 페이지 새로고침 (Ctrl+R 또는 F5)
2. 브라우저 개발자 도구 확인 (F12)
   - Application/Storage > Cookies
   - `guild.sid` 쿠키가 있는지 확인
3. 쿠키가 없다면 브라우저 쿠키 설정 확인

### 퀘스트 제출이 안 됩니다
**증상**: Submit 버튼을 눌러도 제출되지 않음

**해결 방법**:
1. 필수 입력 항목을 모두 채웠는지 확인
2. 브라우저 콘솔에서 에러 메시지 확인 (F12 > Console)
3. 페이지 새로고침 후 다시 시도
4. 로그인 상태 확인 (로그아웃 후 재로그인)

### 경험치가 지급되지 않습니다
**증상**: 퀘스트 제출 후 경험치가 증가하지 않음

**확인 사항**:
1. 제출이 성공적으로 완료되었는지 확인
2. 프로필 페이지에서 총 경험치 확인
3. 페이지 새로고침 (캐시 문제일 수 있음)

### 리더보드가 로드되지 않습니다
**증상**: 리더보드 페이지가 빈 화면으로 표시됨

**해결 방법**:
1. 페이지 새로고침
2. 네트워크 연결 확인
3. 브라우저 콘솔에서 에러 확인 (F12)
4. 잠시 후 다시 시도 (서버 일시적 문제일 수 있음)

### 페이지가 느립니다
**증상**: 페이지 로딩이나 반응이 느림

**해결 방법**:
1. 인터넷 연결 속도 확인
2. 브라우저 캐시 삭제
3. 다른 탭들을 닫기 (메모리 확보)
4. 브라우저 재시작

## 자주 묻는 질문 (FAQ)

### Q1: GitHub 계정이 없으면 사용할 수 없나요?
**A**: 네, 현재는 GitHub OAuth만 지원합니다. GitHub 계정 생성은 무료입니다. https://github.com/join

### Q2: 퀘스트를 여러 번 완료할 수 있나요?
**A**: 네, 같은 퀘스트를 여러 번 제출할 수 있고, 매번 경험치를 받습니다.

### Q3: 제출한 퀘스트를 취소할 수 있나요?
**A**: 현재는 제출 취소 기능이 없습니다. 신중하게 제출해주세요.

### Q4: 레벨의 최대치가 있나요?
**A**: 아니요, 레벨 제한은 없습니다. 무한정 레벨업이 가능합니다.

### Q5: 퀘스트는 누가 만드나요?
**A**: 현재는 관리자가 퀘스트를 생성합니다. 향후 사용자 퀘스트 생성 기능이 추가될 예정입니다.

### Q6: 모바일에서 사용할 수 있나요?
**A**: 네, 모바일 브라우저에서도 사용 가능합니다. 다만 최적화된 경험을 위해 데스크톱 사용을 권장합니다.

### Q7: 계정을 삭제하고 싶어요
**A**: 현재는 자체 삭제 기능이 없습니다. 관리자에게 문의해주세요.

### Q8: 다른 사용자와 채팅할 수 있나요?
**A**: 현재는 채팅 기능이 없습니다. 향후 업데이트에서 추가될 예정입니다.
